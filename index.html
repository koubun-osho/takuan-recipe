塩抜きの方法
HOW TO DESALT

基本水に浸す方法

  1. たくあんを食べやすい大きさに切ります（薄切りにすると早く塩が抜けます）。
  2. ボウルにたっぷりの水を入れ、切ったたくあんを浸します。
  3. 30分〜2時間ほど置き、途中で味見をしてお好みの塩加減に調整します。
  4. 水気をしっかり絞ってから、お召し上がりください。

時短薄い塩水を使う方法

  1. 水500mlに対して小さじ1/2ほどの塩を溶かし、薄い塩水を作ります。
  2. 切ったたくあんを薄い塩水に浸します。
  3. 20分〜1時間ほどで、均一に塩が抜けます。
なぜ塩水？ 真水だと表面だけ急に塩が抜けて中は塩辛いままになりがちです。薄い塩水を使うと、浸透圧の差がゆるやかになり、中まで均一に塩抜きできます。
コツ 浸しすぎると水っぽくなり、たくあん本来の風味が失われます。こまめに味見をして、少し塩気が残るくらいで引き上げるのがおすすめです。

たくあんレシピ集
TAKUAN RECIPES
1
たくあんチャーハン
ポリポリ食感がやみつきの定番
▶

材料（2人分）

  • ご飯 … 茶碗2杯分
  • たくあん（みじん切り）… 60g
  • 卵 … 2個
  • 長ねぎ（みじん切り）… 1/3本
  • ごま油 … 大さじ1
  • 醤油 … 小さじ1
  • 塩・こしょう … 少々

作り方

  1. フライパンにごま油を熱し、溶き卵を入れてさっと炒める。
  2. 半熟のうちにご飯を加え、強火でパラパラになるまで炒める。
  3. たくあんとねぎを加えてさっと炒め合わせる。
  4. 醤油を鍋肌から回し入れ、塩・こしょうで味を調えて完成。
たくあんに塩気があるので、醤油と塩は控えめに。塩抜きなしでもOKです。
2
たくあんの天ぷら
サクサク×ポリポリの新食感
▶

材料

  • たくあん（8mm厚の輪切り）… 8枚
  • 天ぷら粉 … 50g
  • 冷水 … 80ml
  • 揚げ油 … 適量

作り方

  1. たくあんを軽く塩抜きして、キッチンペーパーで水気をしっかり拭く。
  2. 天ぷら粉を冷水で溶いて衣を作る。
  3. たくあんに衣をつけ、170℃の油でカラッと揚げる（約2分）。
  4. お好みで天つゆや塩でどうぞ。
外はサクサク、中はポリポリ。お酒のおつまみにもぴったりです。
3
たくあんタルタルソース
ピクルスの代わりに和風アレンジ
▶

材料

  • たくあん（みじん切り）… 40g
  • ゆで卵 … 2個
  • マヨネーズ … 大さじ3
  • パセリ（みじん切り）… 少々
  • レモン汁 … 小さじ1

作り方

  1. ゆで卵をフォークで粗くつぶす。
  2. みじん切りにしたたくあん、マヨネーズ、パセリ、レモン汁を加えて混ぜる。
  3. フライやチキン南蛮にたっぷりかけてどうぞ。
ピクルスの代わりにたくあんを使うことで、コリコリとした食感と独特の風味が加わります。
4
たくあんとクリームチーズの和え物
混ぜるだけ、日本酒にもワインにも
▶

材料

  • たくあん（細切り）… 50g
  • クリームチーズ … 50g
  • かつお節 … ひとつかみ
  • 醤油 … 少々
  • お好みで大葉の千切り

作り方

  1. クリームチーズを1cm角に切り、室温に少し戻しておく。
  2. 細切りにしたたくあんと合わせ、かつお節をまぶす。
  3. 醤油をほんの少したらして和えれば完成。
切って混ぜるだけの簡単おつまみ。クリームチーズのコクとたくあんの歯ごたえが絶妙です。
5
たくあんの卵焼き
お弁当にもぴったりの彩りおかず
▶

材料

  • たくあん（みじん切り）… 30g
  • 卵 … 3個
  • 砂糖 … 小さじ1
  • だし汁 … 大さじ1
  • サラダ油 … 適量

作り方

  1. 卵を溶きほぐし、砂糖・だし汁・たくあんのみじん切りを混ぜる。
  2. 卵焼き器に薄く油をひき、卵液を3回に分けて流し入れ、巻いていく。
  3. 粗熱が取れたら食べやすい大きさに切って完成。
たくあんの黄色が断面に映えて、お弁当が華やかに。食感のアクセントにもなります。

GOOD TO KNOW

保存方法

開封後はラップでぴったりと包み、冷蔵庫で保存してください。
空気に触れると風味が落ちやすくなります。
冷蔵保存で約2〜3週間が目安です。お早めにお召し上がりください。

「たくあん」と曹洞宗のご縁

たくあん漬けの名前の由来には諸説ありますが、江戸時代の曹洞宗の高僧・沢庵宗彭（たくあんそうほう）禅師にちなむという説が広く知られています。

沢庵禅師は、質素な食事を大切にする禅の教えの中で、大根の漬物を工夫して作ったと伝えられています。

蓮城院も同じ曹洞宗の寺院です。禅の精神を受け継ぎ、境内で丁寧に手作りしたたくあんを、どうぞ味わっていただけましたら幸いです。

おいしい切り方のコツ

そのまま食べるなら5mm程度の厚さがおすすめ。歯ごたえを楽しめます。
料理に使う場合は、用途に合わせてみじん切り・細切り・輪切りに。
斜めに薄く切ると、断面が大きくなって味がよく馴染みます。
